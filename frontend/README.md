# Frontend (Phase 1)

Vite + React + Tailwind + React Router.

## Prerequisites
- Node.js 18+

## Setup
```bash
cd frontend
npm install
```

## Environment
Create `frontend/.env` from `frontend/.env.example`.

```bash
cp .env.example .env
```

## Run
```bash
npm run dev
```

App will be available at `http://localhost:5173`.
